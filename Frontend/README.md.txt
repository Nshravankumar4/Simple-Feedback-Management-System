# Simple Feedback Management System

## Backend

### Setup
1. Install dependencies:
npm install


2. Run the server:
npm run dev



### Structure
- `src/app.ts`: Express server setup
- `src/server.ts`: Server entry point
- `src/routes/feedback.routes.ts`: Feedback routes
- `src/controllers/feedback.controller.ts`: Feedback controller
- `src/services/feedback.service.ts`: Feedback service

## Frontend

### Setup
1. Install dependencies:
npm install


2. Run the app:
npm start



### Structure
- `src/components/FeedbackForm.tsx`: Form to submit feedback
- `src/components/FeedbackList.tsx`: List of feedback entries
- `src/App.tsx`: Main application component
- `src/worker.ts`: Web worker for fetching data
